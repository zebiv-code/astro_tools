# Astro Tools

Historical astronomical computation tools ported from classic Sky & Telescope BASIC programs. Calculate sunrise/sunset, moon phases, holidays, meteor showers, and more.

**[Live Demo](https://zebiv.com/astro-tools/)**

## Tools

- **Sunrise/Sunset** — Times and azimuths for any location and date
- **Moon Phases** — New Moon, First Quarter, Full Moon, Last Quarter dates
- **Calendar** — Monthly calendar grid for any month/year
- **Holidays** — Easter, Passover, Rosh Hashanah, Yom Kippur
- **Meteor Showers** — Peak dates for major annual showers
- **Julian Date** — Calendar to/from Julian Day conversion
- **Sidereal Time** — Greenwich Mean Sidereal Time calculation

## Tech

- Vanilla HTML/CSS/JavaScript
- No dependencies, no build step
- Algorithms from Sky & Telescope magazine (1985-1994)
- Meeus lunar phase algorithm
